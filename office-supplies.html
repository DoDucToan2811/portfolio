Office Supplies System

A comprehensive web application for digitizing office supplies request and approval workflows, developed during my internship.

Backend Repository Frontend Repository

Project Description and Goals

My primary responsibility during the internship was to develop an Office Supplies System. The aim was to digitize the workflows for requesting, approving, and summarizing office supply transactions, thereby improving transparency and reducing paperwork.

The system provides a complete solution for managing office supplies across different organizational levels, from employees to department leaders, super leaders, and finance staff.

Technology Stack

  • Frontend: Vue.js
  • Backend: ASP.NET Core RESTful API
  • Database: Microsoft SQL Server with Entity Framework ORM
  • Security: JWT Authentication

Key Features and Requirements

  • Conducting requirement analysis and reviewing business documentation
  • Developing modules for request form creation and submission
  • Automating summary report generation
  • Implementing user roles: Employees, Department Leaders, Super Leaders, Finance Staff
  • Real-time tracking and department-specific usage reports
  • Improved communication between requesters and approvers

Business Workflow

Office Supplies Request Process

  • Step 1: Employees log in and submit requests
  • Step 2: Department Leaders review and approve or reject the request
  • Step 3: Finance Managers validate requests based on budget constraints and usage norms

Office Supplies Summary Process

  • Step 1: Finance Staff generate summary forms from all approved requests
  • Step 2: Summary forms are submitted to Directors for approval
  • Step 3: Upon approval, the procurement or release process is initiated

Project Goals

  • Digitize all request processes for office supplies
  • Provide real-time tracking and department-specific usage reports
  • Improve communication between requesters and approvers
  • Reduce manual workload and improve cost efficiency

What I Learned

This internship project provided invaluable hands-on experience in full-stack web development. I gained expertise in building modern Single Page Applications (SPA) using Vue.js on the frontend, while developing robust RESTful APIs with ASP.NET Core on the backend.

Working with Entity Framework ORM and SQL Server deepened my understanding of database design and management. I learned to implement secure authentication and authorization using JWT tokens, ensuring proper role-based access control across different user levels.

Beyond technical skills, this project enhanced my ability to analyze business requirements, design efficient workflows, and translate complex business processes into functional software solutions. I developed strong problem-solving skills while addressing real-world challenges in enterprise application development.
