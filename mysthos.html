What I Learned

Developing Mysthos Mysterium, a library web app, enhanced my knowledge of web application architecture and dynamic user interfaces. I mastered the use of React.js for building modular and reusable components, enabling a seamless user experience. The project introduced me to Phaser.js, where I explored its capabilities for creating interactive and visually engaging features for the app. Additionally, I learned how to utilize Node.js for backend development to handle server-side logic and user data management efficiently. This project significantly improved my ability to combine user-focused design with functional and scalable code.
